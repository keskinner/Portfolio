NFL Quiz

This NFL Quiz challenges what you know about the NFL. The quiz will tell you if you got a question right or wrong after each answer is submitted. It also keeps a score throughout the quiz and gives you a total score at the end. What’s the highest score you can get?

Brewery Finder

Brewery Finder is an app that lets you search for different types of breweries in a given city. Search results are then shown on a map with markers. You can change the breweries that are shown on the map by the following types: brewery/pubs, miro breweries, large breweries, or regional breweries.

List of technologies used:

  • HTML
  • CSS
  • Javascript
  • JQuery
  • Google Maps API Documentation
  • Open Brewery DB API Documentation
